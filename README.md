# 🧠 Projeto Quiz - C# Windows Forms

Um aplicativo de Quiz interativo desenvolvido em **C#** utilizando a tecnologia **Windows Forms**. O projeto conta com diferentes categorias, sistema de pontuação e uma interface gráfica amigável.

## 🚀 Funcionalidades

- **Múltiplas Categorias de Estudo:** Sistema intuitivo de seleção de matéria para iniciar o quiz.
- **Interface Intuitiva com Tabs:** Organização clara entre "Questões de Estudo", "Resultados" e "Gestão de Utilizadores".
- **Gestão de Acesso:** Suporte para diferentes níveis de utilizador (ex: Admin, Padrão).
- **Feedback Visual:** Uso de ícones, gráficos e backgrounds personalizados para cada tema.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** C#
- **Framework:** .NET (Windows Forms)
- **IDE:** Visual Studio
